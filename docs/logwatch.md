## Logwatch Setup

Instal logwatch <br>
`apt-get install logwatch`

Go to config location <br>
`/usr/share/logwatch/default.conf/`

Make a backup of the config file <br>
`cp logwatch.conf backup.conf`

Edit the file according to references
