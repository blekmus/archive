## List Deduplication

``` py
>>> some = [1, 2, 3, 4]
>>> filtered = set(some)
>>> filtered
# {1, 2, 3}
```

## Zero Padding

``` py
>>> i = 2
>>> padded = f"Three {i:03d}"
>>> padded = '{foo:03d}'.format(foo=i)
>>> padded
# Three 002
```

## Remove Whitespace

``` py
>>> text = "  we are   god   "
>>> formatted = "".join(text.split())
>>> formatted
# wearegod
```
